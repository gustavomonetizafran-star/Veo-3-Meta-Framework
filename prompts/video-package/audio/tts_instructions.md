# Instrucciones para Generar VO en Español (Femenino)

## Generación de VO

Para generar voces en español utilizando el archivo `voice_script_es.md`, se recomienda seguir el siguiente proceso de segmentación por escena:

1. **Chunking por Escena**: Divide el texto en segmentos claros que correspondan a cada escena. Nombrar los archivos siguiendo el formato `sceneXX_vo.mp3` (por ejemplo, `scene01_vo.mp3`).
2. **Configuraciones de Calidad**: Se sugiere utilizar una frecuencia de muestreo de 48 kHz y formatos como MP3, WAV o FLAC.
3. **Notas sobre Licencias y Costos**: Asegúrate de consultar las licencias de cada servicio utilizado y tener en cuenta los costos asociados.

## Recomendaciones
